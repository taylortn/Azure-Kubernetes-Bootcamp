# Lab 2 - Azure Identity & RBAC

## Objective
Understand and implement Azure Role-Based Access Control (RBAC)

## Key Concepts
- Azure Active Directory (Azure AD / Entra ID)
- Users and Groups
- Role-Based Access Control (RBAC)
- Scope (Subscription, Resource Group, Resource)

## Tasks
- Create a new user
- Assign RBAC role
- Test permissions

## Outcome
Demonstrated ability to control access to Azure resources using RBAC
